# 📊 RetailIQ – Phone Retail Sales Analytics System

RetailIQ is a Python-based sales analytics application developed to automate Excel-based sales reporting for a mobile phone retail business. The application processes sales data, generates business insights, visualizes revenue trends, and stores data in SQLite for future analytics.

---

## 🚀 Features

- Read sales data from Excel
- Calculate total revenue and profit
- Analyze top-selling brands and models
- Detect low-stock inventory
- Generate Excel summary reports
- Create revenue visualization charts
- Store sales data using SQLite

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/sanskriti18/RetailIQ.git
cd RetailIQ
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
